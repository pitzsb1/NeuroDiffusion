### 배경 및 필요성 

- **데이터 희소성**

뇌전증 발작 데이터는 정상 데이터에 비해 양이 매우 적으며, 특히 환자 개개인마다 발작 파형이 상이하여 일반화된 모델 구축이 어려움.

- **개인차의 한계**

범용적인 통계치만으로는 특정 환자에게만 나타나는 미세한 발작 전조 증상을 포착하는 데 한계가 있음.

- **생성 AI의 가치**

최근 시계열 Diffusion 모델의 발전으로, 실제와 통계적으로 유사하면서도 새로운 파형을 생성하여 희귀 데이터 부족 문제를 해결할 수 있게 됨.

### 프로젝트 목표

- **환자 맞춤형 데이터 증강**

소량의 실제 발작 데이터를 바탕으로 Diffusion 모델을 통해 정교한 합성 EEG 데이터를 대량 생성.

- **딥러닝 기반 고정밀 분류**

증강된 데이터를 학습에 활용하여 기존 머신러닝 대비 발작 감지 재현율(Recall)을 획기적으로 향상.

- **실제적 불균형 해소**

클래스 불균형 문제를 해결하여 누락 없는 발작 감지 시스템의 기틀 마련.

### 주요 기능 및 특징

- **Time-Series Diffusion Engine**

1D 시계열 신호의 주파수와 위상을 보존하며 데이터를 생성하는 SSSD 또는 DiffWave 기반 생성 엔진 구축.

- **Conditional Data Synthesis**

환자 ID 및 질환 유형을 조건(Condition)으로 입력하여 목표하는 특정 패턴의 데이터를 선별적으로 생성.

- **성능 검증 파이프라인 (TSTR)**

합성 데이터로 학습(Train on Synthetic)하고 실제 데이터로 테스트(Test on Real)하여 생성된 데이터의 유효성을 정량적으로 검증.

### 일정 및 계획

**1주차 | 데이터 프로파일링 및 생성 모델 설계**

환자별 발작 데이터 추출 및 정문화.

1D-Diffusion 모델 아키텍처 설계 및 베이스라인 학습.

**2주차 | 데이터 증강 및 품질 최적화**

환자 맞춤형 합성 뇌파 생성 및 실제 데이터와의 통계적 유사성 비교.

생성된 데이터를 포함한 학습 데이터셋 구축.

**3주차 | 고도화 모델 학습 및 벤치마크**

증강 데이터 활용 전/후의 분류 성능(F1-Score, Recall) 비교 분석.

하이퍼파라미터 튜닝을 통한 최적의 증강 비율 도출.

**4주차 | 결과 분석 및 최종 리포트**

시각화 지표를 통한 생성 데이터의 타당성 입증.

환자 가족용 '실시간 스크리닝' 시나리오에 따른 최종 성능 보고.

### 기대효과

- **희귀 질환 연구의 확장**

데이터가 부족한 희귀 뇌전증 케이스에서도 AI 모델의 성능을 보장할 수 있음.

- **진단 정확도 극대화**

데이터 증강을 통해 모델이 더 많은 발작 패턴을 학습함으로써 오진율을 낮추고 재현율을 높임.

### 결론

병원 내 대규모 데이터에만 의존하던 기존 방식에서 탈피하여, 환자 개개인의 소량 데이터만으로도 고성능 맞춤형 모델을 구축할 수 있는 '데이터 증강 엔진'으로서의 가치를 증명할 것임.

### 기존 프로젝트의 문제점 개선

- **데이터 불균형 및 희소성 해결**

참고 프로젝트는 주어진 Kaggle 데이터셋의 분포 내에서만 학습이 가능하여, 데이터가 부족한 희귀 발작 유형이나 특정 환자의 개인차를 반영하는 데 한계가 있음. 본 프로젝트는 Diffusion 모델을 통한 데이터 합성으로 학습 데이터의 양적/질적 한계를 극대화하여 극복함.

- **전처리 병목 현상 제거**

참고 프로젝트는 EEG를 스펙트로그램 이미지로 변환하는 복잡한 수학적 전처리가 필수적이며, 이 과정에서 신호의 미세한 위상(Phase) 정보가 손실될 수 있음. 본 프로젝트는 1D Raw Signal을 직접 처리하여 정보 손실을 최소화함.

### 기존 프로젝트의 성능 개선 (기술적 차이)

- **고정적 특징 vs 동적 확률 분포 학습**

참고 프로젝트가 이미 존재하는 이미지 패턴을 인식하는 수준이라면, 본 프로젝트는 뇌파의 확률적 분포(Generative Distribution) 자체를 학습함. 이는 모델이 단순히 '외운 패턴'을 찾는 것을 넘어, 발생 가능한 '변칙적 패턴'까지 예측하게 함.

### 실제 환경 개선 및 반영 제안

본 딥러닝 프로젝트는 고도화된 생성 AI 기술을 활용하여 '전문 의료 현장 및 임상 연구' 수준으로 타겟을 확장함. 또한 실제 발작 데이터가 부족한 신약 개발 현장이나 임상 시험에서, 특정 환자의 뇌파 스타일을 모사한 합성 데이터를 제공하여 연구의 가속화를 도움.
