### 배경 및 필요성 

진단의 복잡성: 뇌파(EEG) 데이터는 다채널 시계열 신호로, 숙련된 신경과 전문의조차 판독에 상당한 시간과 에너지를 소모

데이터 해석의 한계: 원본 시계열 신호는 노이즈가 많고 전극 간의 상관관계를 직관적으로 파악하기 어려워 단순한 알고리즘으로는 분류 정확도가 떨어짐

자동화 시스템의 요구: 응급 상황(발작 등)에서 신속하고 객관적인 진단을 보조할 수 있는 고성능 AI 모델의 개발이 시급함

### 프로젝트 목표

고효율 데이터 표현: 방대한 EEG 시계열 데이터를 딥러닝 모델이 학습하기 최적화된 '4채널 스펙트로그램' 형태로 압축 및 변환

분류 정확도 극대화: 전문의들의 투표 분포(Consensus)를 모사하여 6가지 주요 뇌 활동(발작, LPD, GPD 등)에 대한 예측 확률의 오차(KL Divergence)를 최소화

강력한 일반화 성능 확보: 환자별 특성에 치우치지 않는 범용적인 뇌파 분석 알고리즘을 구축

### 주요 기능 및 특징

도메인 특화 전처리 (Bipolar Montage): 뇌의 공간적 구조를 반영하여 LL, LP, RP, RR 4개 영역의 전위차 신호를 생성, 뇌 활동의 국소적 특징을 강화

다채널 스펙트로그램 설계: 1D 신호를 2D 이미지로 변환 후, 4개 구역의 정보를 하나의 텐서로 결합하여 CNN(EfficientNet)이 공간적/주파수적 특징을 동시에 학습시킴

노이즈 저감 기술: Wavelet Transform 및 정규화(Normalization)를 적용하여 신호의 신뢰도 증대

하이브리드 앙상블: 이미지 기반 딥러닝(CNN)과 통계 기반 머신러닝(CatBoost)의 결과를 결합하여 예측 안정성 증대

### 기대효과

의료 의사결정 지원: 전문의의 판독 시간을 단축하고, 주관적 판단에 따른 오류를 줄여 진단의 객관성을 확보 가능

기술적 확장성: 본 프로젝트에서 제안하는 '시계열-이미지 변환 프레임워크'는 심전도(ECG), 진동 데이터 분석 등 다른 시계열 도메인에도 적용 가능

### 결론

본 프로젝트는 스펙트로그램 기반의 CNN 학습과 통계적 머신러닝의 앙상블을 통해 전문가의 투표 분포를 정밀하게 모사함으로써, 복잡하고 불확실한 뇌 활동 판독 과정에 객관적이고 신뢰도 높은 가이드라인을 제시함. 이는 의료 AI가 나아가야 할 방향인 '도메인 지식과 데이터 사이언스의 유기적 결합'을 실현한 모델로서, 향후 다양한 생체 신호 분석 분야의 표준 프레임워크가 될 수 있음.

### 기존 프로젝트의 문제점 개선 | 라벨 불확실성 대응

전문가들의 투표가 갈리는 '모호한 데이터'에 대해 단순 평균을 내기보다, 신뢰도 가중치(Confidence Weighting)를 부여하여 모델이 더 명확한 진단 기준을 학습하도록 손실 함수를 정교화

### 기존 프로젝트의 성능 개선 (기술적 차이) | 의사 결정 최적화 (TTA: Test Time Augmentation)

추론 시에도 데이터를 미세하게 변형(Shift, Flip)하여 여러 번 예측한 뒤 평균을 내어 단일 예측보다 안정적이고 정확한 결과를 도출

### 실제 환경 개선 및 반영 제안

실시간 모니터링 시스템(Edge AI) 최적화: 병상 곁에서 즉각적인 피드백이 가능하도록 모델을 경량화(TensorRT, ONNX 변환)하여 저지연(Low-latency) 추론 시스템 구축 가능

설명 가능한 AI(XAI) 도입: 의사가 모델의 판단 근거를 확인할 수 있도록, 스펙트로그램의 어느 부위가 이상 징후로 판단되었는지 보여주는 Heatmap(Grad-CAM) 기능을 추가하여 진단 신뢰도를 높임
