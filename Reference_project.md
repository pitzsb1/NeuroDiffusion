# Fashion-MNIST

### 딥러닝 참고 프로젝트 제안서

Info.

총 7만 개의 의류로 구성된 흑백 이미지를 학습하여, 어떠한 의류의 이미지를 입력하였을 때 학습된 데이터를 토대로 카테고리를 판단하는 모델

- 점과 직선을 넘어 의류의 '특징'을 보고 판단
- 실제 산업에서 의류 카테고리를 분류하는 기초가 될 수 있음

GOAL: 의류 이미지 데이터를 학습하여 카테고리 분류

DATA: Zalando Research (Fashion-MNIST)

### 참고 프로젝트 선정 이유

본 프로젝트에서는 대량의 의류 이미지 데이터를 토대로 카테고리 분류를 학습하여 새로운 이미지를 입력받아도 어떠한 종류의 의류인지 분석하는 모델임. 앞으로 진행될 프로젝트가 이미지 학습 모델임을 감안했을 때 좋은 참고 자료가 될 것이라고 판단하여 선정하게 됨.

## 원본 데이터 | Zalando Research

원본 데이터 출처: Zalando Research (Fashion-MNIST)

데이터 크기 및 구성
- Total: 70,000개의 이미지
- Train Set: 60,000개 (모델 학습용)
- Test Set: 10,000개 (모델 성능 검증용)

이미지 규격(**독립 변수 x**): 28 x 28 픽셀의 흑백 이미지

레이블(**종속 변수 y**): 0부터 9까지의 정수로 된 10개 카테고리 (T-shirt, Trouser 등)

## 데이터 전처리

### 정규화

이미지의 각 픽셀 값은 0(검은색)에서 255(흰색) 사이의 값을 가짐. 이를 0과 1 사이의 실수로 변환


모든 픽셀 값을 255.0으로 나누기 (x / 255.0)

-> 값이 너무 크면 학습 과정에서 계산값이 튀거나 느려질 수 있는데, 0~1 사이로 맞추면 경사하강법(Gradient Descent)시 수렴 속도가 훨씬 빨라짐

# 모델링 분석

### 사용된 알고리즘: CNN (Convolutional Neural Network)

주요 레이어

- Conv2D: 이미지의 특징(선, 면, 패턴)을 추출하는 필터 역할

- MaxPooling2D: 이미지의 크기를 줄이면서 중요한 정보만 남기는 압축 과정을 수행

- Dropout: 학습 중 무작위로 뉴런을 꺼서 특정 데이터에만 과도하게 최적화되는 '과적합'을 방지

- Dense (Fully Connected): 추출된 특징들을 종합하여 최종적으로 10개의 카테고리 중 하나로 분류

## 알고리즘 선정 이유

1. 공간적 구조 유지
- 일반 알고리즘(예: RandomForest): 이미지를 1열로 길게 늘어뜨린 데이터로 취급함. 

- CNN: 이미지를 격자 형태 그대로 받아들임. 필터가 이미지 위를 훑으며 인접한 픽셀 간의 관계를 파악하므로 옷의 형태나 구조적 특징을 훨씬 잘 잡아냄.
   
1. 이동 및 변형에 대한 강인함
옷 사진이 정중앙에 있지 않고 살짝 왼쪽으로 치우쳐 있어도 CNN의 필터는 이미지 전체를 훑기 때문에 동일한 소매나 단추 패턴을 찾아낼 수 있음.

1. 계층적 특징 학습 (Hierarchical Learning)
낮은 층에서는 점과 선을 보고, 높은 층에서는 소매와 다리 모양을 봄. Fashion-MNIST처럼 풀오버와 코트의 미세한 차이를 구분해야 하는 문제에서는 이런 단계별 특징 추출이 필수적임.

## 성능 평가 분석

### 사용된 성능 평가 지표
이 프로젝트는 '분류(Classification)' 문제이므로, 회귀 분석에서 쓰는 RMSE 같은 지표 대신 분류 전용 지표를 사용

- Accuracy (정확도)

전체 10,000개의 테스트 이미지 중 모델이 정답을 맞힌 비율

프로젝트에서는 약 91~92%의 정확도를 목표로 함. 10벌 중 9벌은 정확히 어떤 옷인지 맞힐 수 있음.

- Loss (손실 함수 - Cross Entropy)
  
모델이 내놓은 예측값과 실제 정답 사이의 거리를 수학적으로 계산한 값

학습이 잘 될수록 이 Loss 값은 0에 가깝게 떨어짐. 모델이 학습을 진행하면서 이 수치가 낮아지는 것을 모니터링하여 올바른 방향으로 학습하고 있는지 확인.

### 한계 극복을 위한 시도 및 최적화 전략
단순히 층을 쌓는 것만으로는 부족하기 때문에, 모델의 안정성과 성능을 높이기 위한 몇 가지 하이퍼파라미터와 기법들을 포함

- Dropout (과적합 방지)
모델이 학습 데이터만 외워버리면 과적합되기 때문에, 무작위로 뉴런의 연결을 끊는 Dropout(0.25~0.5) 레이어를 추가. 이는 모델이 특정 특징에만 의존하지 않고 더 보편적인 규칙을 찾도록 강제함.

- Batch Normalization (데이터 진정시키기)
레이어 사이사이에 데이터의 분포를 고르게 맞추는 과정을 추가하여 학습을 안정적으로 진행시키고, 초기값 설정에 덜 예민해지게 함.

## 최종 성능 요약

본 프로젝트는 평가 지표였던 정확도 92%를 달성함. 또한 Dropout을 적재적소에 사용하여 과적합을 효과적으로 방지함. 하지만 최신 모델들이 정확도 96~99%를 기록하는 것에 비하면 기초적인 수준이기에 실용 가능성이 떨어짐.
